Chúc các bạn có nhiều kiến thức trên kênh học trực tuyến này.

Bởi Admin Root - Wednesday, 9 June 2021, 8:53 AM

Các bạn sinh viên đang làm bài tập tiểu luận ?, file của bạn rất lớn > 10MB, làm cách nào để có thể nén được file pdf nhưng vẫn đảm bảo chất lượng file ?. Trung tâm Quản trị Hệ thống xin giới thiệu các bạn công cụ nén file hiệu quả sau, Link nén file online: https://tools.pdf24.org/vi/nen-pdf , ngoài ra nếu bạn muốn dùng offline vẫn có thể download phần mềm này vể để sử dụng được luôn nhé.
